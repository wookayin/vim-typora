# vim-typora

Vim ❤️ [Typora][typora]


Usage
-----

```
:Typora
```

to open the current file in Typora (macOS or UNIX only).

Vim buffers and Typora windows will synchronize upon every save.

On UNIX, `typora` must be within `PATH`.
